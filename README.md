## Support Platform

- macOS 10.13+ (High Sierra+)
- Windows (Developing...)
- Linux (Developing...)

## Features

- Multiple account save & Apply to notion app.
- Touchbar support (At notion app focused)
- Menu bar support (At notion app focused)
